# SAGE Care Monitoring

SAGE Care Monitoring is a frontend-only Vue/Vite prototype for DON, Provider, and CNA clinical coordination workflows.

## Requirements

- Node.js 20.19 or newer
- npm

## Setup

```sh
npm install
```

## Run Locally

```sh
npm run dev
```

The app runs at `http://127.0.0.1:5174/` by default.

## Useful Commands

```sh
npm run typecheck
npm run build
npm run serve
```

The runnable app lives in `artifacts/mobile`.
